Admin Login Mystery is developed for gives a user extra options to customize the wp-admin or wp-login or website admin page customize.

== DESCRIPTIONS ==
ADMIN LOGIN MYSTERY is a plugin to gives users a extra-options to customize the login page. After installing this plugin you can get an option from your website admin parent of settings LIKE SETTINGS => ADMIN LOGIN MYSTERY. Then you can get options to add image, height, width, etc. You can also add some additional css to your login page. Also, You can update settings without reloading the page. We're still working on it to user satisfaction. If you need some help or need some additional functionality on our plugin you can get support from us.
